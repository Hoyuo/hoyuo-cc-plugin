# prompt-engineering

Claude Code 시스템 프롬프트 아키텍처 기반 프롬프트 엔지니어링.

31개 Claude Code 시스템 프롬프트 원문을 참조하여 검증된 패턴 기반으로 프롬프트를 작성하고 분석한다.

## 사용법

```
/prompt-engineering write <목적>        # 프롬프트 작성
/prompt-engineering analyze <프롬프트>   # 프롬프트 분석/개선
/prompt-engineering patterns            # 핵심 패턴 보기
/prompt-engineering reference [번호]    # 원문 참조 (00~30)
```

## 참조 원문 (references/)

31개의 Claude Code 시스템 프롬프트 원문을 포함. 시스템 프롬프트, 에이전트 프롬프트, 스킬 프롬프트, 도구 프롬프트 등 다양한 유형을 커버한다.
