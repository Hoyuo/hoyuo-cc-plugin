# harness-engineering

Claude Code 하네스 엔지니어링.

하네스 구축과 실행을 단계별 스킬로 수행한다.

## 스킬

### 구축 단계

| 스킬 | 설명 |
|------|------|
| init | 프로젝트 컨텍스트 분석, 하네스 구축 시작 |
| step | 하네스 개별 스텝 설계/구성 |
| validate | 구축된 하네스 구조/동작 검증 |
| score | 하네스 품질 점수화, 개선점 제시 |

### 실행 단계

| 스킬 | 설명 |
|------|------|
| plan | 하네스 활용 작업 실행 계획 수립 |
| exec | 계획에 따라 하네스 실행 |
| eval | 실행 결과 분석, 피드백 생성 |
| update | 평가 결과 반영하여 하네스 개선 |

### 도움말

| 스킬 | 설명 |
|------|------|
| help | 사용 가능한 스킬과 워크플로우 안내 |
